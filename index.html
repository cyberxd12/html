Semantica

diferente do html 4, o html 5 foca mais em significados na ultilização das funções, exemplo:
Eu moro na

Rua Adenaldo Ferreto - N39 - Parque Lagoa Subae

Nesse caso vemos que tem uma tag que destaca o endereço ao inves de usar outras ferramentas para essa funções(mas ainda é possivel ultilizar, mas essa forma habilita coisas novas como interações com outros software).

Principais Formatações

negrito

nessa frase temos um negrito usando a tag < b >(não Semantica)
nesta frase temos um destaque usando a tag <strong >(Semantica)

italico

nesta frase temos um italico com a tag < i >(não Semantica)
nesta frase temos uma emfasase usando a tag < em >(Semantica)

Texto Marcado

Texto Grande e Pequeno

Listas

Listas ordenadas

lista ordenada é feita usando o < ol > e para adicionar o objeto na lista usa o < li >, ultilizando o type (1, A, a, I, i) pode mudar o tipo de marcação da lista e ultilizando o start muda a marcação inicial

Lista não Ordenada

lista ordenada é feita usando o < ul > e para adicionar o objeto na lista usa o < li >, ultilizando o type (disc, circle, square) pode mudar o tipo de marcação da lista

Mais Sobre Listas

criar uma lista apenas para usar como exemplo:

  1. html
  2. css
  3. javaScript
  4. python
  • mySql
  • Power BI
  • excel
  • World

Links

Links externos

Usando a tag a e alterando o href="", pode adicionar tanto um link para um arquivo interno dentro do projeto quanto para um arquivo/pagina que esta na web, no caso de links externos recomendado que ultilize junto com a teg a o complemento(target="_blank" rel="external") para que abra o link em outra aba.
ex:
acesse meu Git Hub: click aqui
OBS: o link esta estilizado com o css pois tenho um pequeno conhecimento previo e já apliquei.

Links internos

Mesma coisa do link externo mas com o complemento(target="_self" pra abrir na propria pagina ou deixa sem nada mesmo) e com o caminho um pouco mais simples e com a condição de estar na mesma pasta do projeto inicial index. ex: usando um arquivo ja usado para teste, as metas.

Links para Download

Links de download são buscados do projeto e contem os complementos (download="" com o nome do arquivo+tipo e o type="" com o midia type certo para cada arquivo.)
ex:

  • meu curriculo.pdf
  • meu curriculo.zip
lista com os types: iana.org

COMEÇO DO CSS

gradiente

comando css: background-image: linear-gradient(to top "lado inicial", white "cor 1", blue "cor 2");
background-attachment: fixed "definir para toda a pagina";
obs: quantidade de cor definida pelo programador

padding

comando css: padding: 10px "tamanho do padding";
obs: padding é o espaçamento entre o conteudo e o elemento
obs: padding-top, padding-bottom, padding-left, padding-right

margin

comando css: margin: 10px "tamanho do margin";
obs: margin é o espaçamento entre o elemento e o elemento
obs: margin-top, margin-bottom, margin-left, margin-right

border

comando css: border: 1px "tamanho do border" "tipo do border" "cor do border";
obs: border é o espaçamento entre o elemento e o elemento
obs: border-top, border-bottom, border-left, border-right
